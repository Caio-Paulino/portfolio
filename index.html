Meu nome é Caio Paulino e meu primeiro contato com programação foi em 2019, no meu curso técnico de informática. Apesar de já ter interesse na área, ainda não dado a devida atenção a esse estudo. Somente em 2023, após entrar no curso de Análise e Desenvolvimento de Sistemas, passei a estudar com verdadeiro afinco para me tornar um programador.

Atualmente estudando Java com Spring Boot e ReactJS, sou um desenvolvedor Full Stack e gosto de sempre estar aprendendo coisas novas.

Habilidades

ReactJS: é uma biblioteca JavaScript de código aberto utilizada para construir interfaces de usuário, especialmente em aplicações web de página única. Criada pelo Facebook, ela permite a criação de componentes reutilizáveis e proporciona uma renderização eficiente e dinâmica da interface, resultando em uma experiência de usuário mais fluida.

Java: é uma linguagem de programação orientada a objetos, conhecida por sua portabilidade e desempenho. Utilizada em uma ampla gama de aplicações, desde desenvolvimento web até aplicativos móveis e sistemas corporativos, Java é robusta, segura e possui uma vasta comunidade de desenvolvedores e bibliotecas.

Spring Boot: é um framework Java que simplifica a criação de aplicações autônomas e prontas para produção. Ele oferece uma abordagem opinativa para configuração e permite que os desenvolvedores criem rapidamente serviços e aplicações web robustas e escaláveis, com uma configuração mínima.

Git: é um sistema de controle de versão distribuído, amplamente utilizado para rastrear mudanças em arquivos de código-fonte durante o desenvolvimento de software. Ele permite que múltiplos desenvolvedores trabalhem simultaneamente em um projeto, facilita a colaboração e ajuda a gerenciar o histórico de versões de maneira eficiente.

MySQL: é um sistema de gerenciamento de banco de dados relacional de código aberto, utilizado para armazenar e gerenciar grandes volumes de dados. Reconhecido por sua velocidade, confiabilidade e facilidade de uso, é amplamente adotado em diversas aplicações web, desde pequenos projetos até grandes sistemas corporativos.
